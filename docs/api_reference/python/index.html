Python API reference

Model access

  • pitasc.model : Contains functionality for creating and modifying pitasc models and loading existing models from XML.

  • pitasc.model_interface : Convenience tools for accessing and modifying pitasc models more easily.

  • pitasc.parameter_model : Contains functionality for creating and modifying pitasc model elements (used by pitasc.model).

  • pitasc.model_element : Convenience tools for accessing and modifying pitasc model elements more easily (used by pitasc.model_interface).

Runtime access

  • cppitasc.logging : Logging functionality used by the pitasc runtime.

  • cppitasc.runtime : Runtime access for executing pitasc applications and runtime data exchange.
